# DocAppoint - Doctor Appointment Manager

A modern and responsive doctor appointment booking platform where users can browse doctors, book appointments, manage bookings, and update profiles securely.

## 🔗 Live Website

👉 https://assingment-9-umber.vercel.app

---

## 📌 Project Features

- 🔐 Secure Authentication system using Better Auth with JWT/session
- 👨‍⚕️ Browse and view detailed doctor information dynamically
- 📅 Book doctor appointments with user-friendly modal/form
- 🛠️ Update and delete appointments instantly without page reload
- 👤 User profile management with live update functionality
- 🔎 Search appointments by doctor name
- 📱 Fully responsive design for Mobile, Tablet, and Desktop
- ⚡ Loading spinner and custom toast notifications
- 🚫 Custom 404 error page for invalid routes
- 🎨 Modern UI/UX with clean layout and consistent styling
